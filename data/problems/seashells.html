Sally's sorted seashells by the shell sizes. Seeing sea shell sizes, state sorting success.

Input

Each test case consists of two lines. The first line will contain a single integer N - the number shells on the shore. The second line will contain N positive integers. The ith integer represents the size of the ith shell in the list.

Output

If the sea shells are sorted - that is, that no shells are followed by a shell of a smaller size - print "Sorting Successful". Otherwise, print "Sad Sorting, Sally"

Constraints

Both the number of shells and the size of each shell will be a positive integer less than 500,000.
