Something that classical composers used to compose into their compositions was a fugue. There are many types of fugues. Repeating a string of notes but in a different octave(the same notes are played but in a different pitch) is one type of fugue. Repeating a string of notes backwards is also a fugue. Combining multiple fugue techniques is also a fugue.

You need to write a program that can detect if two phrases of notes are one of these fugues.

  1. The notes are the same but the entire string is shifted to a single different octave.
  2. The notes are reversed.
  3. Both a and b.

For this problem, notes will be represented by an integer. There are only 8 notes (1-8), but many octaves. So 1 is the same note as 9 and 17, etc. A "0" means that no note is played at that time.

Input

The first input will be the number of notes in each string. The second line will be a string of space seperated integers between 0 and 72 representing the first string of notes. The third line will be a string of space seperated integers between 0 and 72 representing the second string of notes. Each line will have up to 8 integers, and each line will only have numbers from one octave or 0's.

Output

If the second note string is a fugue of the first note string: print "true"; if it's not a fugue: print "false".

4
1 2 3 4
9 10 11 12

true

4
1 2 1 2
2 1 2 1

true
